# claude-code-openclaw-skill

A standalone OpenClaw skill package for routing work to Claude Code through ACP sessions, with prompts shaped by Claude Code official documentation.

## Contents

- `claude-code-openclaw/` - the skill source
- `dist/claude-code-openclaw.skill` - packaged skill artifact

## Skill name

- `claude-code-openclaw`

## Purpose

This skill helps OpenClaw route Claude Code requests with better prompt construction, stronger verification requirements, and cleaner ACP session usage.
